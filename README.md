# Finnish Presidents – Wiki Hits with Retrofit

### Features

-	Display a list of Finnish Presidents with:
-	Name
-	Term of office (start–end)
-	Short description
-	Fetch Wikipedia hits for each president using Retrofit.
-	Click on a president to load Wiki hits individually.
-	Refresh the Wiki hits per president.
-	Loading indicator while fetching data.
-	Fully implemented using Jetpack Compose.
